# dev utility

a brutalist developer utility suite focused on efficiency and speed.

## features

- bulk bin generator (luhn compliant)
- mock checking engine with real time logs
- temporary email inbox via 1secmail api
- interactive 3d card visualizer
- keyboard first navigation via command palette

## technical stack

- next.js 15
- react 19
- tailwind css v4
- gsap 3
- lucide icons

## local environment

npm install
npm run dev
